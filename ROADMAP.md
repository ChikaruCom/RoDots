# Roadmap

## MVP

- `.rdot` document editing and preview
- `date`, `input`, and `meta` commands
- Date row repeat syntax
- Preview-side date editing
- View, split, and edit modes
- `*.view.rdot` startup behavior
- `*.rock.rdot` locked view behavior
- Header/footer gadget zones
- Simple dark/light theme toggle
- Metadata breadcrumbs
- File template save command
- Local path opening
- URL check and cache skeleton
- Cache backup and restore skeleton

## Next

- Real file open/save flow for `.rdot`
- `*.view.rdot` view lock behavior
- Safer source rewrite model for repeated date rows
- Calendar popup UI beyond the native date input
- Config-driven header widget layout from `config.json`
- GitHub Actions for frontend checks
- Rust/Tauri build verification on Windows

## Later

- Standalone HTML export
- Richer file template definitions
- Web cache refresh UI
- Document package portability
- Signed Windows installer
